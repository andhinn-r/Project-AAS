My Project

Project 1

This project focuses on creating an anti-bullying poster using Adobe Illustrator. The purpose is to learn vector design while delivering a message about kindness and respect. During the process, we explore how colors, shapes, and text can work together to create a clear and attractive layout. We practice choosing readable fonts, balancing composition, and highlighting key messages so people understand the importance of stopping bullying. This activity teaches us how visual design can influence emotions and encourage positive behavior. Through the project, we gain creative skills and learn to communicate meaningful messages through digital artwork.

Project 2

This project is about tracing character images using Adobe Illustrator to improve our vector illustration skills. We learn basic tools such as the Pen Tool, Shape Tool, and Direct Selection Tool to recreate outlines and colors accurately. The task trains our precision in adjusting curves, controlling anchor points, and matching details from a reference image. We also practice applying fills, simple shading, and clean line work so the final illustration looks neat and consistent. Even though we follow an existing picture, the goal is to produce our own vector version. This activity builds patience, observation, and confidence in digital drawing.

See Project

Project 3

This project focuses on recreating the Instagram logo using Adobe Illustrator. The aim is to learn basic vector design by building shapes, controlling proportions, and applying gradients similar to the original logo. Through this activity, we explore how simple geometric forms can become a professional icon. We practice using the Shape Tool, Align panel, Gradient settings, and Stroke adjustments to achieve a clean result. Recreating a well-known logo teaches us to observe details carefully and understand the importance of consistency in branding. This project helps strengthen our technical skills and prepares us for future logo-making tasks.
